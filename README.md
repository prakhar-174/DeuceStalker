# Hi there, I'm Prakhar Tiwari! 👋
*(aka DeuceStalker)*

Welcome to my personal portfolio repository! I built this space to showcase my journey as a **Frontend Developer, Problem Solver, and Web Craftsman**. 

Instead of a standard resume, I wanted to build an experience that truly reflects my personality and design sensibilities. I drew heavy inspiration from the dynamic, glitchy, and comic-book aesthetics of the **Spider-Verse**—combining deep void backgrounds, glassmorphism, and incredibly smooth animations into a highly interactive web experience.

## ✨ What's inside my portfolio?

- **My Work**: A showcase of the projects I've poured my heart into, presented with slick GSAP scroll animations and frosted glass cards.
- **My Skills**: A custom-built, interactive node-based skills network that visually maps out my technical stack.
- **My Story**: A little bit about who I am, what drives me to write code every day, and my journey so far.
- **Interactive Details**: I coded a custom WebGL/Canvas hero section and a pure CSS morphing side-panel navigation because I absolutely love sweating the small details.

## 🛠️ How I Built It

- **Core Engine**: [React.js](https://reactjs.org/) powered by [Vite](https://vitejs.dev/) for an incredibly fast development environment.
- **Styling**: Vanilla CSS Modules. I prefer having total, granular control over my custom properties, glowing accents, and UI layouts without relying on heavy frameworks!
- **Motion**: A blend of [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://greensock.com/gsap/) to handle the buttery-smooth transitions and scroll reveals.
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/) (Feather pack).

## 🚀 Peek under the hood

If you want to poke around my code and see how I built the animations or the glass UI, feel free to clone it and run it locally:

```bash
# Clone the repository
git clone https://github.com/prakhar-174/DeuceStalker.git

# Navigate into the project
cd DeuceStalker

# Install dependencies
npm install

# Start the local server
npm run dev
```
